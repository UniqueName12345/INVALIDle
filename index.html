Invalidle

Wordle is a game where you try to make words out of letters. This is wordle but invalid words are guessed instead.

The game is played by typing words into the text box. If the word is valid, it is added to the list of invalid words. If the word is invalid, it is added to the list of valid words.

The game wins when the list of invalid words is full.

The game loses when the list of valid words is full.

Controls:

  • Type a word into the text box.
  • Press the submit button to add the word to the list of invalid words.
  • Press the reset button to clear the text box.
  • Press the reset button to clear the list of invalid words.
  • Press the reset button to clear the list of valid words.
